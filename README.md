# 🐱 LexiCat – 喵喵单词屋

A cute and interactive English vocabulary learning app built with Streamlit.  
Designed to make word learning engaging, intuitive, and enjoyable 🐾

---

## ✨ Features

- 🧠 **Context-based learning**
  - Learn words through real example sentences
  - Fill-in-the-blank exercises

- 🔊 **Pronunciation support**
  - Click to hear word pronunciation
  - Sentence-level audio after answering correctly

- 🐟 **Gamified progress**
  - Feed your cat by answering correctly
  - Daily goal system (10 fish per day)

- 🐾 **Smart feedback system**
  - Encouraging responses for wrong answers
  - Progressive hints after multiple attempts
  - Random penalty (lose 1–3 fish after 3 mistakes, adding challenge)

- 📘 **Review cards**
  - Word + IPA + meaning
  - Syllable breakdown
  - Common collocations

## 🚀 Run Locally

### 1. Clone the repo

```bash
git clone https://github.com/JingmiaoLi/LexiCat.git
cd lexicat
```
### 2. Install dependencies
pip install -r requirements.txt

### 3. Run the app
streamlit run app.py
